# 10-Minuten-Aktivierung – Präsentation (Reveal.js)



## Überblick

Diese Präsentation behandelt die **10-Minuten-Aktivierung in der Pflege von Menschen mit Demenz**.  
Sie wurde mit **Reveal.js** umgesetzt und dient als visuelle Unterstützung für Unterricht oder Vortrag.

## Inhalte

- Grundlagen der 10-Minuten-Aktivierung
- Aufbau einer Aktivierungseinheit
- Praktische Beispiele aus der Pflege
- Bewegungstherapie
- Musik in der Begleitung von Menschen mit Demenz
- Eigene Reflexion und Praxisbezug

## Technologien

- HTML
- CSS
- JavaScript
- Reveal.js
